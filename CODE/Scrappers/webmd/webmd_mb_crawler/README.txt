To run a WebMD Message Boards Browser Automation based crawler, follow these steps:

1. Install node.js on your machine if it is not already installed.
2. Navigate to the "webmd_mb_crawler" directory.
3. Run the command "npm install" to install the application dependency.
4. Start the application by running the command "npm start app.js".
5. To crawl forum post links, access the following link from your browser: "http://localhost:8080/webmd/msgboards/crawllinks".
6. To crawl forum posts, access the following link from your browser: "http://localhost:8080/webmd/msgboards/crawlposts".
Before running this step, make sure that the links have been crawled and saved under the "links_json" directory.


Alternatively, you can also run the "runmbcrawler.sh" file to execute the WebMD Message Boards Browser Automation based crawler.
However, make sure to install node.js and run the "npm install" command to install the necessary dependencies before running the
script.
